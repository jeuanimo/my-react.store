## How to install
1. Clone this repository
```bash
git clone https://github.com/jeuanimo/my-react.store.git
```
2. Go into project
```bash
cd my-react.store.git
```
3. Install dependencies
```bash
npm install
```

4. Run project
```bash
npm run dev
```
